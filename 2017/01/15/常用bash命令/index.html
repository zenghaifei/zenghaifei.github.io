一、文件与文件夹操作

1、进入文件目录

  • 查看当前所在目录路径 pwd (print working directory 的简写)
  • 根据相对路径（相对当前路径）进入一个目录 cd folderName/subfolderName/
  • 根据绝对路径进入一个目录 cd /folderName/subfolderName/
  • 返回上一级或多级目录 cd ..(cd ../..) 注：”..”代表上一级目录，“.”代表当前目录
  • 进入home目录 cd
  • 返回上一次目录跳转前所在目录 cd -

    2、查看目录文件信息

  • 列出当前目录的所有可见文件名 ls （”ls”代表单词“list”）
  • 列出当前目录下的所有文件名（含隐藏文件，隐藏文件的文件名以”.”开头） ls -a （”-a”代表“all”）
  • 详细列出当前目录下的所有可见文件信息 ls -l (“-l”代表“long”)
  • 递归列出当前目录下的所有可见文件及文件夹下的文件 ls -R
  • 用系统默认软件打开当前目录下的一个文件 open [fileName]
  • 用指定软件打开一个文件 [softwareName] [fileName] 如：用vim打开当前目录下的text.txt文件vi test.txt

3、创建删除文件与文件夹

  • 创建一个新的文件夹（当前目录下） mkdir [fileName] （”make directory”的简写）
  • 删除一个文件夹（当前目录下）
    1. 删除一个空文件夹：rm -r [folderName] （”rm”为”remove”的简写,“-r”代表单词“recursive”）
    2. 递归删除一个文件夹（一个个需要确认） rm -ri [folderName] （其中“-i”代表单词“inspect”）
    3. 强制删除一个文件夹 rm -rf [folderName] (“-rf”中的“f”代表单词“force”)
  • 创建一个新的文件(当前目录下) touch [fileName]
  • 删除一个文件(当前目录下)
    1. 删除一个文件 rm [fileName]
    2. 需要确认的方式删除一个文件 rm -i [fileName]
    3. 强制删除一个文件 rm -f [fileName]

3、加快命令操作速度的快捷键

  • 自动补全文件或文件名 输入文件或文件名的前一个或几个字符按tab键
  • 一键擦除当前行的录入信息 ctrl + u
  • 显示前一次（或后一次）执行的命令 ↑（后一次↓）
  • 清空当前屏幕 clear
