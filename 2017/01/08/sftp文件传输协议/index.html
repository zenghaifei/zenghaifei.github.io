sftp的应用

发表于 2017-01-08   |  

简介

sftp是Secure File Transfer Protocol的缩写，为一种安全的文件传输协议，基于安全可靠的ssh协议（默认端口号为22）加密传输，因而安全性非常好。常用的文件传输协议还有ftp、ftps，在此简要对比一下这几种协议的差别：

FTP FTPS SFTP
安全性 极差（明码传输) 好(FTP+SSL) 好(加密，完整性校验)
传输速度 快 一般 一般
端口号 双端口号 双端口号 单端口

以上对比可知，sftp由于其安全与易用性已经成为文件传输的首选方案。
