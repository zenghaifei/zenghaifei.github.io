常用bash命令

发表于 2017-01-15   |  

一、基本知识

  • “~”在shell中代表home目录
  • “..”为上一级目录的链接
  • “.”为当前目录的链接
  • 文件名以“.”开头的为隐藏文件
  • 查看当前目录路径 pwd (print working directory 的简写)

二、文件与文件夹操作

1、进入文件目录

  • 根据相对路径（相对当前路径）进入一个目录 cd folderName/subfolderName/
  • 根据绝对路径进入一个目录 cd /folderName/subfolderName/
  • 返回上一级或多级目录 cd ..(cd ../..)
  • 进入home目录 cd
  • 返回上一次目录跳转前所在目录 cd -

2、查看目录文件信息

  • 列出当前目录的所有可见文件名 ls（”ls”代表单词“list”）
  • 列出当前目录的所有可见文件名并以最近修改时间排序 ls -t
  • 列出当前目录下的所有文件名（含隐藏文件）ls -a （”-a”代表“all”）
  • 参数可以叠加使用，如ls -alt 代表的是“-a，-l，-t”的叠加效果
  • 递归列出当前目录下的所有可见文件及文件夹下的文件 ls -R
  • 用系统默认软件打开当前目录下的一个文件 open [fileName]
  • 用指定软件打开一个文件 [softwareName] [fileName] 如：用vim打开当前目录下的text.txt文件vi test.txt
  • 详细列出当前目录下的所有可见文件信息 ls -l (“-l”代表“long”)

3、创建删除文件与文件夹

  • 创建一个新的文件夹（当前目录下） mkdir [fileName] （”make directory”的简写）
  • 删除一个文件夹（当前目录下）
    1. 删除一个空文件夹：rm -r [folderName] （”rm”为”remove”的简写,“-r”代表单词“recursive”）
    2. 递归删除一个文件夹（一个个需要确认） rm -ri [folderName] （其中“-i”代表单词“inspect”）
    3. 强制删除一个文件夹 rm -rf [folderName] (“-rf”中的“f”代表单词“force”)
  • 创建一个新的空文件(当前目录下) touch [fileName]
  • 删除一个文件(当前目录下)
    1. 删除一个文件 rm [fileName]
    2. 需要确认的方式删除一个文件 rm -i [fileName]
    3. 强制删除一个文件 rm -f [fileName]

4、移动拷贝文件

  • 在当前目录下复制一个文件为新的文件名 cp [file] [newfile]
  • 复制一个文件到指定目录 cp [file] [dir] (其中dir可以是相对路径，也可以是绝对路径)
  • 重命名一个文件 mv [fileName] [new filename]
  • 移动一个文件到一个文件夹
    1. mv [fileName] [dir] (目标文件夹含同名文件直接覆盖)
    2. mv -i [fileName] [dir] (目标文件夹含同名文件提醒是否覆盖)

三、指令与程序操作

  • 以管理员权限执行一个命令 sudo [command] (“sudo”代表“super user do”)
  • 后台执行一条命令 [command] &
  • 后台执行一条命令，并使其在session断开后依然执行 nohup [commandName] &
    “nohup ”代表“no hang up ”, 如：nohup java -jar XXX.jar & 代表在后台一直执行一个jar包
    注：通常在shell中执行的后台运行程序在session结束后就自动关闭了，如果需要session结束后程序依然在后台执行，则加上”nohup”
  • 显示前一次（或后一次）执行的命令 ↑（后一次↓）
  • 显示最后的n条历史命令 history [n]
  • 交互式的搜索历史指令 ctrl r
  • 查看程序或指令是否在运行：ps -ef|grep [commandName]（支持名称的部分匹配，展示的结果第二列的数字为pid）
  • 显示当前正在后台运行的进程：top （按q退出）
  • 退出进程
    1. 强制退出当前进程 ctrl + c
    2. 退出指定进程 kill -15 [pid]
    3. 强杀一个进程 kill -9 [pid]
  • 执行shell脚本文件（.sh为后缀）步骤：
    1. 使脚本可执行：chmod a+x [name].sh
    2. 执行脚本： ./[name].sh
  • [command A] ; [command B] 执行完A后执行B，无论A是否执行成功
  • [command A] && [command B] 执行A，当A执行成功后才执行B

四、输入输出与重定向

  • echo 输出到屏幕上(会在输出的最后一行加上一个新的空行)
  • printf 输出到屏幕上(不会添加空行)
  • cat 提取一个文件的内容并显示到屏幕上（如：cat 1.txt代表将1.txt中的文本内容打印到屏幕上）
  • 重定向输出到一个文件中
    1. 覆盖原文件（如果输出的文件存在的话）”>”, 如：echo “Hello” > hello.txt
    2. 附加到原文件内容后（如果待输出的文件已经存在的话）“>>”, 如：printf “hope” >> hello.txt
  • 重定向输入 “<”, 如：cat < lakes.txt 将lakes.txt作为输入源重定向给cat程序
  • 管道“|”——连接一个程序的输出到另一个程序的输入
    如：cat 1.txt | wc | cat > islands.txt，将cat 1.txt的输出连接为wc的输入源，再将wc的输出作为输入源连接到cat命令

五、其它

  • 查看本机ip: ifconfig -a
  • 自动补全文件或文件名 输入文件或文件名的前一个或几个字符按tab键
  • 一键擦除当前行的录入信息 ctrl + u
  • 切换前后两个单词 按Esc键 再按T键
  • reset Resets the terminal display
  • pbcopy < [file] Copies file contents to clipboard
  • pbpaste Paste clipboard contents
  • pbpaste > [file] Past clipboard contents into file, pbpaste > paste-test.txt
